# Who Wants To Be A Millionaire - Enhanced Edition

A modern reimagining of the classic trivia game with exciting new features, multiple game modes, and competitive multiplayer gameplay.

## Features

### Game Modes
- **Classic Mode** - Traditional single-player progression through difficulty tiers
- **Multiplayer Mode** - Compete against friends in real-time
- **Timed Challenge** - Answer as many questions as possible within the time limit
- **Survival Mode** - One wrong answer and you're out

### Difficulty Levels
- **Easy** - Warm up with accessible questions
- **Medium** - Test your general knowledge
- **Hard** - Challenge yourself with difficult trivia
- **Expert** - For the true trivia masters

### Enhanced Lifelines
- **50/50** - Remove two incorrect answers
- **Phone a Friend** - Get advice from an AI assistant
- **Ask the Audience** - See poll results from community answers
- **Double Dip** - Answer twice, keep the higher result
- **Peek** - Secretly view the correct answer once per game

### Additional Features
- Customizable question categories
- Detailed statistics and leaderboards
- Progressive prize multipliers
- Streak tracking
- Sound effects and background music

## Getting Started

1. Extract the game files
2. Run the executable
3. Select your game mode and difficulty
4. Start playing and climb toward the million!
